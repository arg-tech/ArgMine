Tool Options

1. Input a new source text and a new other text. Each will be passed to the Argument Mining Framework. Text should be in this format (including line breaks):

Speaker: Text text text.

Speaker: Text text text.

2. Input a new source text and compare with an existing argument map or set of argument maps for Intertextual Correspondence (ITC). This requires the Other AIF Map Mode checkbox to be checked. Map numbers should be given as:

10672,10673,10674

3. Input an existing argument map as source and get ITC connections with another existing source. Requires selecting both AIF map mode checkboxes.

4. Input a new source text and get ITC connections with Hansard the UK parliamentary record. Only requires source text and the Hansard ITC selection.

5. Input an existing source map and get ITC connections with Hansard the UK parliamentary record. Only requires source maps in comma separated list format and the Hansard ITC selection.
